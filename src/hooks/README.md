# Custom React Hooks

This directory contains custom React hooks for reusable logic.

## Purpose

Custom hooks encapsulate reusable stateful logic that can be shared across components.

## Examples

- `useLocalStorage` - Persist state in localStorage
- `useDebounce` - Debounce values
- `useMediaQuery` - Respond to media queries
- `useFetch` - Data fetching
- `useForm` - Form handling
- `useAuth` - Authentication logic

## Guidelines

- Prefix all hooks with "use"
- Keep hooks focused and single-purpose
- Document parameters and return values
- Consider edge cases and error handling
- Write tests for complex hooks
